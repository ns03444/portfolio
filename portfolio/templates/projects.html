Python Package: Network Automation Utilities.

There are many Python packages available that can be used to aid network automation developers in their work. One such package is the "netutils" package, which contains a variety of utilities that can be used to automate various network tasks. This package is designed to make it easy for developers to access and use common network tools and protocols, such as ping, traceroute, SSH, and SNMP, within their Python scripts. With the "netutils" package, developers can easily write scripts to automate tasks such as network configuration, monitoring, and troubleshooting. Whether you are a beginner or an experienced network automation developer, the "netutils" package can help you streamline your workflow and increase your productivity.

Django Application: Content Aggregator.

Developed a “content aggregator” for technology related stock market news and performance data which was deployed to the cloud using Git and Python Anywhere that dynamically updates once every hour. I began by using the Beautiful Soup (bs4) and Requests dependencies to web-scrape unformatted data from various websites. Then, the front end was developed using the Django frontend framework along with Bootstrap HTML and CSS to present my scraped content, along with an SQLite3 backend database for handling, serving, updating, etc. the content of the site. Next, I manipulated/created calculated columns in a pandas data-frame of open/close prices for the S&P 500 to create interactive candlestick charts with Bokeh embedded into a Bootstrap template. Finally, I conducted an in-depth financial analysis of the 3 largest wireless network carriers in America by using the yFinance and Matplotlib python libraries. The yFinance library was implemented to collect the close price, volume, volatility, market capitalization, Beta, price-to-earnings ratio, dividend yield. Then, I integrated the data into a single data-frame and used Matplotlib to create various visualizations (line graphs, bar charts, histograms). Last, I used the visualizations to provide an educated conclusion (not investment advice).
